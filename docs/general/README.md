# General semantic conventions

This document defines general Semantic Conventions adopted from the
[OpenTelemetry Semantic Conventions](https://github.com/open-telemetry/semantic-conventions/tree/main/docs/general).

The following general Semantic Conventions are defined:

* [Naming](naming.md): Naming guidelines for semantic conventions.
* [Attribute requirement level](attribute-requirement-level.md): Requirement levels for attributes.
* [Metric requirement level](metric-requirement-level.md): Requirement levels for metrics.
* [Recording Errors](recording-errors.md): Conventions for recording errors.
* [Semantic convention groups](semantic-convention-groups.md): How groups work in semantic conventions.

For the full set of general conventions (attributes, spans, metrics, logs, events),
see the upstream [OpenTelemetry semantic-conventions](https://github.com/open-telemetry/semantic-conventions/tree/main/docs/general).
